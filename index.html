Generate n x n Square

An interactive grid generator with customizable start, end, and obstacles.

5 x 5 Square:

1. Click on a cell to set up the start grid as green, the end grid as red, or an obstacle as gray.

2. The number of obstacles is limited to n-1 (Limit: 4).

Hint: Click cycling logic → Start, then End, then Obstacle, then Empty.
